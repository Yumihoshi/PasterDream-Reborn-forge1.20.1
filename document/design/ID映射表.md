# ID 映射表

> 源模组旧 ID → 重置模组新 ID 对照表  
> 每次搬运方块/物品时**必须**登记，无论是否修改 ID

---

## 使用说明

- 搬运一个方块/物品时，先在此表中查找是否已登记
- 未登记的，按实际搬运情况填写
- 变更说明须简洁明了：`不改` / `重命名` / `合并变体` / `拆分` 等

---

|-------|-------|---------|------------|
