# 目录结构组织

> Forge 1.20.1 模组项目标准目录结构 — PasterDream Reborn

---

## 项目根目录

```
PasterDream-Reborn-forge1.20.1/
├── build.gradle              # Gradle 构建脚本（模组依赖、运行配置、数据生成）
├── settings.gradle           # Gradle 设置（插件仓库、工具链）
├── gradle.properties         # 构建属性（MC 版本、模组 ID、版本号等）
├── gradle/                   # Gradle Wrapper
├── gradlew / gradlew.bat     # Gradle 启动脚本
├── libs/                     # 第三方本地 jar（Curios、GeckoLib、JEI 等）
├── NOT_MODIFY/               # ⚠️ 旧版模组参考（只读，严禁修改任何文件）
│   ├── reference/            #   旧版模组源码参考（net.pasterdream），不参与编译
│   ├── reference-resources/  #   旧版模组资源参考（assets + data）
│   └── old-recipes/          #   旧版配方（按产物 ID 重命名，搬运时检索用）
├── document/                 # 项目文档（非代码）
│   ├── rule/
│   │   └── program/          # 程序规范文档
│   │       ├── 目录结构组织.md
│   │       ├── 架构规范.md
│   │       └── 程序规范文档.md
│   ├── design/               # 策划与设计文档
│   │   ├── 策划文档.md
│   │   ├── 命名规范文档.md
│   │   ├── 新内容列表.md
│   │   ├── ID映射表.md
│   │   ├── mapping.json      # ID 映射 JSON（由 generate_id_mapping.py 自动生成）
│   │   └── generate_id_mapping.py
│   └── reference/            # 参考文档
│       ├── 源项目结构分析.md
│       ├── 原模组创造物品栏.md
│       └── 重写项目结构说明.md
├── run/                      # 开发环境运行目录
└── src/                      # 源代码与资源
    ├── main/
    │   ├── java/com/pasterdream/pasterdreammod/
    │   └── resources/        # 静态资源（手动维护）
    └── generated/            # 数据生成器输出（自动生成，不可手动编辑）
```

## 关键目录说明

| 目录 | 说明 | 手动/自动 |
|------|------|----------|
| `init/` | 注册中心（Blocks、Items、Tabs、ParticleTypes 等） | 手动编写 |
| `world/block/` | 有特殊机制的客制化方块类及子包 | 手动编写 |
| `world/item/` | 有特殊机制的客制化物品类及子包 | 手动编写 |
| `world/fluid/` | 流体实现类 | 手动编写 |
| `world/fluidtype/` | 流体类型类 | 手动编写 |
| `world/behavior/` | 方块/物品行为逻辑 | 手动编写 |
| `worldgen/` | 世界生成 registry bootstrap（ConfiguredFeature、PlacedFeature） | 手动编写 |
| `worldgen/biome/` | 群系 + 群系修改器 bootstrap | 手动编写 |
| `worldgen/dimension/` | 维度类型 + 噪声设置 + LevelStem bootstrap | 手动编写 |
| `client/` | 客户端渲染注册、HUD | 手动编写 |
| `client/particle/` | 粒子行为类 + 粒子提供者注册 | 手动编写 |
| `datagen/common/` | 数据生成器核心 Provider | 手动编写 |
| `datagen/lang/` | 语言文件生成器 | 手动编写 |
| `datagen/util/` | 数据生成器工具类 | 手动编写 |
| `tag/` | 方块/物品标签常量 | 手动编写 |
| `network/` | 网络数据包（含子包） | 手动编写 |
| `capability/` | 能力系统（MeltDreamEnergy、SAN） | 手动编写 |
| `command/` | 调试命令（含子命令） | 手动编写 |
| `recipe/` | 通用配方框架 | 手动编写 |
| `helper/` | 辅助工具 | 手动编写 |
| `component/` | GUI 组件 | 手动编写 |
| `compat/jei/` | JEI 配方兼容插件 | 手动编写 |
| `mixin/` | Mixin 占位 | 手动编写 |
| `src/main/resources/assets/` | 纹理、粒子定义、静态 JSON | 手动维护 |
| `src/generated/resources/` | 数据生成器输出 | 自动生成（runData），**不可手动编辑** |
