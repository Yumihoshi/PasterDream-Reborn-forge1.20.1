# 目录结构组织

> Forge 1.20.1 模组项目标准目录结构 — PasterDream Reborn

---

## 项目根目录

```
PasterDream-Reborn-forge1.20.1/
├── build.gradle              # Gradle 构建脚本（模组依赖、运行配置、数据生成）
├── settings.gradle           # Gradle 设置（插件仓库、工具链）
├── gradle.properties         # 构建属性（MC 版本、模组 ID、版本号等）
├── gradle/                   # Gradle Wrapper
├── gradlew / gradlew.bat     # Gradle 启动脚本
├── libs/                     # 第三方本地 jar（Curios、GeckoLib、JEI 等）
├── NOT_MODIFY/               # ⚠️ 旧版模组参考（只读，严禁修改任何文件）
│   ├── reference/            #   旧版模组源码参考（net.pasterdream），不参与编译
│   ├── reference-resources/  #   旧版模组资源参考（assets + data）
│   └── old-recipes/          #   旧版配方（按产物 ID 重命名，搬运时检索用）
├── tools/                     # 开发辅助工具（ID 映射、NBT 结构转换）
│   ├── convert_structure.py   #   NBT 结构方块转换工具源码
│   ├── generate_id_mapping.py #   映射表 → JSON 生成工具源码
│   ├── lookup_id.py           #   映射查询工具源码
│   ├── 操作手册.md             #   工具集操作手册
│   └── dist/                  #   编译产物（纳入版本管理）
│       ├── convert_structure.exe
│       ├── generate_id_mapping.exe
│       └── lookup_id.exe
├── document/                 # 项目文档（非代码）
│   ├── rule/
│   │   └── program/          # 程序规范文档
│   │       ├── 目录结构组织.md
│   │       ├── 架构规范.md
│   │       ├── 程序规范文档.md
│   │       └── 容器配方平衡系统.md
│   ├── design/               # 策划与设计文档
│   │   ├── 策划文档.md
│   │   ├── 命名规范文档.md
│   │   ├── 新内容列表.md
│   │   ├── ID映射表.md        # ID 映射数据源（Markdown 表格）
│   │   └── mapping.json       # 由 tools/generate_id_mapping.exe 自动生成
│   └── reference/            # 参考文档
│       ├── 源项目结构分析.md
│       ├── 原模组创造物品栏.md
│       └── 重写项目结构说明.md
├── run/                      # 开发环境运行目录
└── src/                      # 源代码与资源
    ├── main/
    │   ├── java/com/pasterdream/pasterdreammod/
    │   └── resources/        # 静态资源（手动维护）
    └── generated/            # 数据生成器输出（自动生成，不可手动编辑）
```

## 辅助工具目录 `tools/`

开发辅助 Python 脚本及编译产物，用于旧模组 → 新模组的 ID 映射管理和 NBT 结构文件转换。

### 目录结构

```
tools/
├── convert_structure.py          # NBT 结构方块转换工具（交互式 / 命令行两种模式）
├── generate_id_mapping.py        # 从 ID映射表.md 解析生成 mapping.json（保留手动字段）
├── lookup_id.py                  # 查询旧ID → 新ID 映射（支持批量、全列）
├── 操作手册.md                    # 三个工具的完整操作手册
└── dist/                          # 编译产物（纳入版本管理）
    ├── convert_structure.exe
    ├── generate_id_mapping.exe
    └── lookup_id.exe
```

### 数据流

```
document/design/ID映射表.md         ← 人工维护的映射数据源
        │
        ▼
tools/generate_id_mapping.exe       ← 解析 Markdown 表格
        │
        ▼
document/design/mapping.json        ← 自动生成，含 blocks/items/block_properties...
        │
        ├──► tools/lookup_id.exe    ← 命令行快速查询
        │
        └──► tools/convert_structure.exe  ← NBT 结构方块 ID 替换
```

### 重新编译

修改 `tools/` 下的 `.py` 源码后：

```bash
pyinstaller --onefile --name convert_structure   --distpath tools/dist tools/convert_structure.py
pyinstaller --onefile --name generate_id_mapping --distpath tools/dist tools/generate_id_mapping.py
pyinstaller --onefile --name lookup_id           --distpath tools/dist tools/lookup_id.py
```
