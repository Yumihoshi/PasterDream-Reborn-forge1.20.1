# 目录结构组织

> Forge 1.20.1 模组项目标准目录结构 — PasterDream Reborn

---

## 项目根目录

```
PasterDream-Reborn-forge1.20.1/
├── build.gradle              # Gradle 构建脚本（模组依赖、运行配置、数据生成）
├── settings.gradle           # Gradle 设置（插件仓库、工具链）
├── gradle.properties         # 构建属性（MC 版本、模组 ID、版本号等）
├── gradle/                   # Gradle Wrapper
├── gradlew / gradlew.bat     # Gradle 启动脚本
├── changelog.txt
├── CREDITS.txt
├── LICENSE.txt
├── README.txt
├── libs/                     # 第三方本地 jar（Curios、GeckoLib、JEI 等）
├── reference/                # 旧版模组源码参考（net.pasterdream），不参与编译
├── reference-resources/      # 旧版模组资源参考（assets + data）
├── document/                 # 项目文档（非代码）
│   ├── rule/
│   │   └── program/          # 程序规范文档
│   │       ├── 目录结构组织.md
│   │       └── 命名规范文档.md
│   ├── design/               # 策划与设计文档
│   │   ├── 策划文档.md
│   │   ├── ID映射表.md
│   │   ├── mapping.json      # ID 映射 JSON（由 generate_id_mapping.py 自动生成）
│   │   └── generate_id_mapping.py
│   └── reference/            # 参考文档与旧项目数据
│       ├── 源项目结构分析.md
│       ├── 重写项目结构说明.md
│       └── old-recipes/      # 旧版配方（按产物 ID 重命名，搬运时检索用）
├── run/                      # 开发环境运行目录
└── src/                      # 源代码与资源
    ├── main/
    │   ├── java/com/pasterdream/pasterdreammod/
    │   └── resources/        # 静态资源（手动维护）
    └── generated/            # 数据生成器输出（自动生成，不可手动编辑）
```

## 关键目录说明

| 目录 | 说明 | 手动/自动 |
|------|------|----------|
| `init/` | 注册中心（Blocks、Items、Tabs 等） | 手动编写 |
| `world/block/` | 有特殊机制的客制化方块类 | 手动编写 |
| `world/item/` | 有特殊机制的客制化物品类 | 手动编写 |
| `datagen/common/` | 数据生成器核心 Provider | 手动编写 |
| `datagen/lang/` | 语言文件生成器 | 手动编写 |
| `datagen/util/` | 数据生成器工具类 | 手动编写 |
| `client/` | 客户端渲染注册 | 手动编写 |
| `src/main/resources/assets/` | 纹理、静态 JSON | 手动维护 |
| `src/generated/resources/` | 数据生成器输出 | 自动生成（runData），**不可手动编辑** |
